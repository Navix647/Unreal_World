# 强化学习
《Deep Reinforcement Learning for Multi-objective Optimization》
这篇文章提出了一种用强化学习方法解决多目标优化问题的思路。
这篇文章要做的是多目标旅行商问题 即MOTSP *Traveling Salesman Problem*
核心思路：
**1.将多目标优化问题转化成求解多个单目标优化子问题。 （重点）**
一个多目标优化问题通过权重可以转化成单目标优化问题，那么一个权重向量$\mathbf{\omega} =(\omega_1,\omega_2,...,\omega_M)$  这个向量长度是1，就代表一个子问题。 设定N个权重向量就有N个子问题。用这个N个子问题的求解来逼近帕累托前沿**PF（Pareto Front）**



**2.基于邻域参数迁移的策略Neighborhood-based parameter-transfer strategy来快速求解N个子问题**
思路：如果从零求解了一个子问题，那么和它相邻的子问题（即权重向量相近的那一个）可以在前面的基础上进行求解。也就是说除了第一个子问题，剩余子问题的起始点都不是随机，而是可以从原来子问题的神经网络参数开始进行梯度优化，这样收敛速度可以加快。
